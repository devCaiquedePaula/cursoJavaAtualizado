# Matrizes em Java 
- São arranjos bidimensionais ou multidimensionais.
- Cada elemento é acessado por dois índices: linha e coluna.
- Declaração: tipo[][] nomeDaMatriz = new tipo[linhas][colunas];
- Inicialização: nomeDaMatriz[linha][coluna] = valor;
- Exemplo de declaração e inicialização:

  int[][] matriz = new int[3][4]; // 3 linhas e 4 colunas
  matriz[0][0] = 1;
  matriz[0][1] = 2;
  matriz[0][2] = 3;
  matriz[0][3] = 4;
  
- Acesso aos elementos: int valor = matriz[linha][coluna];
- Laços aninhados são frequentemente usados para percorrer matrizes.

- Exemplo de iteração:

  for (int i = 0; i < matriz.length; i++) {
      for (int j = 0; j < matriz[i].length; j++) {
          System.out.println(matriz[i][j]);
      }
  }
